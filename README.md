# Simple-Graph-Sample
A simple graph sample implementation in C.

Used for academy purposes.

Compile:

```
gcc -c -g -o main.o main.c
gcc -c -g -o grafoio.o grafoio.c
gcc -c -g -o grafo.o grafo.c
gcc -o prim *.o
```

Run:

```./prim [Command]```
